üîß How Discord Timestamps Work

Discord timestamps use Unix timestamp format - the number of seconds since January 1, 1970. When you paste a timestamp code like <t:1234567890:F> into Discord, it automatically displays the correct time in each user's local timezone. Our generator converts your date and time input into the proper Unix timestamp and wraps it in Discord's timestamp syntax.

üìù All 7 Discord Timestamp Formats

Discord supports 7 different timestamp formats:

  • t - Short Time (3:30 PM)
  • T - Long Time (3:30:00 PM)
  • d - Short Date (12/31/2024)
  • D - Long Date (December 31, 2024)
  • f - Short Date/Time (December 31, 2024 3:30 PM)
  • F - Long Date/Time (Monday, December 31, 2024 3:30 PM)
  • R - Relative Time (in 2 hours, yesterday, etc.)
